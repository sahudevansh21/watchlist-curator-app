# Personal Watchlist Curator

Welcome to Personal Watchlist Curator, a web application designed to help you meticulously manage your entire movie and TV show collection. Say goodbye to decision paralysis and rediscover your library!

## Features

- **Collection Management**: Log your movies and TV shows with details like genre, actors, mood tags, and runtime.
- **Search & Filter**: Robust options to find exactly what you're looking for.
- **"Surprise Me" Feature**: Get a random watch suggestion based on your specified criteria.
- **Client-Side Storage**: All your data is securely stored in your browser's local storage, ensuring privacy and no external dependencies.

## Setup and Run

1.  **Clone the repository**:
    ```bash
    git clone [repository-url]
    cd watchlist-curator
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the development server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) in your browser.

4.  **Build for production**:
    ```bash
    npm run build
    ```

5.  **Start the production server**:
    ```bash
    npm run start
    ```
